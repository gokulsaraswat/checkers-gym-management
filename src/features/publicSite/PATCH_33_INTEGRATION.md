# Patch 33 - Utility Tools

This bundle adds the first public utility tools track after Patch 32.

## Includes
- public `/tools` route
- BMI quick check
- calorie and macro estimate
- hydration and starter-split suggestions
- homepage, navbar, and footer entry points for the new tools page

## Apply
Copy the files from `overlay/` into the project root after Patch 32 is already applied.

## Notes
- This patch is frontend-first and does not require new backend tables.
- The tools are intentionally lightweight public-site estimates and should support planning, not replace coached assessment.
